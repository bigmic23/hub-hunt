# Hub Hunt 🚀

Hub Hunt is an AI-powered job discovery platform that helps people find jobs locally and globally.

## Overview

Hub Hunt aggregates jobs from multiple sources and ranks opportunities based on user preferences, skills, and behavior.

The platform supports:

- 🌍 Global job discovery
- 🇳🇬 Nigerian job opportunities
- 🏠 Remote jobs
- 🔀 Hybrid jobs
- 🏢 On-site jobs
- ✈️ Visa sponsorship opportunities
- 🤖 AI-powered job recommendations

## Features

- Multi-source job discovery
- AI preference learning
- User behavior-based ranking
- Saved, applied, and rejected job memory
- Resume matching
- Supabase-powered user memory
- Telegram-based job interaction
- Application assistance

## Technology Stack

- Node.js
- Telegraf
- Supabase
- OpenAI API
- JavaScript
- AI ranking engines
