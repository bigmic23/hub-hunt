# Hub Hunt

Hub Hunt is an AI-powered job discovery platform that helps people find jobs locally and globally.

## Features

- 🌍 Global job aggregation
- 🇳🇬 Nigerian job opportunities
- 💼 Visa sponsorship jobs
- 🏠 Remote, Hybrid & On-site jobs
- 🤖 AI-powered job recommendations
- 📄 Resume matching
- 📊 Smart ranking based on user preferences
